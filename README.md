## ✨ Features

| Feature | Description |
|---------|-------------|
| 🎬 **YouTube API** | Fetches 9 latest videos from Apple's official channel |
| 📱 **Product Catalog** | iPhone products from MySQL database |
| 🔍 **Dynamic Routing** | Single product pages with URL parameters |
| 📱 **Responsive Design** | 1 column mobile → 2 columns desktop |
| 🎨 **Modern UI** | Apple-inspired design with Bootstrap |

## 🛠️ Tech Stack

**Frontend:**
- React 19.2
- React Router 7.14
- Vite 8.0
- Bootstrap 5.3
- YouTube Data API v3

**Backend:**
- Node.js 20
- Express.js 5.2
- MySQL 8.0
- CORS
